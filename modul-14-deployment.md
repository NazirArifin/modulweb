# Modul 14 - Deployment & Cloud Basics

Tujuan Pembelajaran: Mahasiswa memahami cara melakukan publikasi aplikasi web agar dapat diakses secara publik.

## Materi
- Environment Variables (`.env`).
- Build proses: `npm run build`.
- Platform deployment: Vercel, Netlify, atau VPS (PM2 & Nginx).

## Praktikum
Lakukan build pada aplikasi Svelte Anda dan Deploy ke Vercel atau GitHub Pages.
