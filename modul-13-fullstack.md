# Modul 13 - Fullstack Integration & Project Management

Tujuan Pembelajaran: Mahasiswa mampu mengintegrasikan frontend Svelte dengan backend Express secara utuh dalam sebuah mini project.

## Deskripsi
Minggu ini fokus pada penggabungan modul-modul sebelumnya menjadi satu aplikasi fungsional.

### Poin Penting:
1. CORS (Cross-Origin Resource Sharing) pada Express.
2. Proxying pada Svelte (Vite).
3. Error handling end-to-end.

## Tugas
Tentukan tema proyek akhir Anda (UAS) dan buatlah struktur Folder awal serta rancangan database.
