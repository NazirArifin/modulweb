- **Minggu 10**: [Authentication & Security](modul-10-auth.md)
- **Minggu 11**: [Frontend Framework (Svelte 5 Basics)](modul-11-svelte-1.md)
- **Minggu 12**: [Advanced Frontend (SPA & State Management)](modul-12-svelte-2.md)
