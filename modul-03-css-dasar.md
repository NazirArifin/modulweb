# Modul 3 - CSS Dasar & Box Model

Tujuan Pembelajaran: Mahasiswa memahami cara kerja CSS, selektor, properti teks, dan konsep Box Model untuk mengatur tata letak elemen.

## Materi

### Selektor & Visual
CSS digunakan untuk mengatur tampilan HTML. Selektor Utama: Tag, Class (`.`), dan ID (`#`). Properti dasar: `color`, `font-family`, `background-color`.

### Box Model
Setiap elemen dianggap sebagai kotak yang terdiri dari:
- **Content**: Isi elemen.
- **Padding**: Jarak antara isi dan border.
- **Border**: Garis tepi.
- **Margin**: Jarak luar antar elemen.

## Praktikum

### 1. Styling Dasar
```css
body { font-family: sans-serif; }
.penting { color: red; font-weight: bold; }
```

### 2. Box Model
```css
.kotak {
  width: 200px;
  padding: 20px;
  border: 2px solid black;
  margin: 10px;
}
```
