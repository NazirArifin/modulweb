# 📘 Roadmap Materi Pemrograman Web

Dokumen ini merangkum alur pembelajaran minggu 2–15 agar mahasiswa mudah melihat progres kompetensi dari **HTML → CSS → JavaScript → Backend → Fullstack → Deployment → Final Project**.

---

## 🗺️ Ringkasan Mingguan

| Minggu | Tema | Capaian Utama |
|---|---|---|
| 2 | HTML (Structure Layer) | Mampu membuat halaman statis terstruktur |
| 3 | CSS Dasar (Pertemuan 1) | Mampu styling dasar dengan selector |
| 4 | CSS Lanjutan (Pertemuan 2) | Mampu membuat layout modern interaktif |
| 5 | JavaScript (Behavior Layer) | Mampu membuat interaksi dinamis di halaman |
| 6 | JavaScript Lanjutan | Mampu mengambil dan menampilkan data API |
| 7 | Backend & Data Persistence (1) | Mampu membangun API dasar |
| 8 | UTS: Perencanaan Final Project | Proposal proyek siap dikerjakan |
| 9 | Backend & Data Persistence (2) | Mampu mengelola data API ke database konsisten |
| 10 | Authentication & Security | Mampu implementasi login & keamanan dasar |
| 11 | Frontend Framework (1) | Mampu membangun UI modular dengan Svelte |
| 12 | Frontend Framework (2) | Mampu membangun SPA terhubung API |
| 13 | Fullstack Project | Mampu integrasi frontend-backend end-to-end |
| 14 | Deployment & DevOps | Aplikasi live dan dapat diakses publik |
| 15 | Final Project & Presentasi | Proyek final siap demo dan dokumentasi rapi |

---

## 📅 Detail Materi per Minggu

## Minggu 2 – HTML (Structure Layer)

### Fokus Materi
- Struktur dasar HTML5
- Tag semantic (header, section, article, footer)
- Form & input
- Table
- Validasi HTML dasar

### Output
Membuat halaman statis terstruktur.

---

## Minggu 3 – CSS (Pertemuan 1)

### Fokus Materi
- CSS Dasar
- Selector
- Box model
- Layout dasar (float, position)

### Output
Mampu membuat styling dasar dan penggunaan selector pada halaman web.

---

## Minggu 4 – CSS (Pertemuan 2)

### Fokus Materi
- Animasi
- CSS Flex
- CSS Grid

### Output
Mampu membangun layout modern dan interaktif dengan CSS.

---

## Minggu 5 – JavaScript (Behavior Layer)

### Fokus Materi
- Variabel, tipe data
- Function
- DOM manipulation
- Event handling
- Form validation

### Output
Interaksi dinamis pada halaman web.

---

## Minggu 6 – JavaScript Lanjutan

### Fokus Materi
- ES6+ features
- Asynchronous JavaScript (Promise, async/await)
- Fetch API
- Local storage
- JSON

### Output
Mengambil dan menampilkan data dari API.

---

## Minggu 7 – Backend & Data Persistence (Pertemuan 1)

### Fokus Materi
- Routing dan struktur project Express
- Controller dan pola request-response
- Koneksi database dari backend
- Model dasar dan API endpoint CRUD sederhana

### Output
Mampu membangun API dasar dengan Node.js, Express, dan koneksi database.

### Catatan Track
Track utama backend adalah Express. Untuk kebutuhan transisi, kelas dapat menggunakan opsi Hybrid Track (Express + PHP) dengan standar penilaian berbasis konsep.

Lihat kebijakan: [hybrid-track-policy.md](hybrid-track-policy.md).

---

## Minggu 8 – UTS: Perencanaan Final Project

### Fokus Materi
- Menentukan tema dan masalah yang ingin diselesaikan
- Menyusun fitur utama (MVP) dan batasan scope
- Mendesain alur aplikasi (user flow) dan wireframe sederhana
- Menentukan pembagian tugas tim dan timeline pengerjaan
- Menyusun kebutuhan teknis (frontend, backend, database, deployment)

### Output
Proposal final project (judul, latar belakang, fitur MVP, wireframe, stack, timeline, dan pembagian tugas).

---

## Minggu 9 – Backend & Data Persistence (Pertemuan 2)

### Fokus Materi
- Pendalaman query dan relasi data
- ORM dan pengelolaan model
- Validasi data dan error handling di layer data
- Integrasi controller dengan database untuk alur fitur end-to-end

### Output
Mampu mengelola data aplikasi secara konsisten dari API ke database.

### Catatan Track
Implementasi pada jalur PHP (jika digunakan) harus memenuhi kontrak API dan capaian konsep yang setara dengan jalur Express.

---

## Minggu 10 – Authentication & Security

### Fokus Materi
- User authentication
- JWT
- Password hashing
- Session & Token

### Output
Implementasi sistem login dan keamanan dasar.

---

## Minggu 11 – Frontend Framework (Pertemuan 1)

### Fokus Materi
- Pengenalan Svelte dan struktur project
- Component
- Props & state
- Event dan komunikasi antar komponen

### Output
Mampu membangun UI modular berbasis komponen dengan Svelte.

---

## Minggu 12 – Frontend Framework (Pertemuan 2)

### Fokus Materi
- SPA concept dan routing dasar
- Lifecycle dan reactive statements
- API integration (fetch data dan render list/detail)
- State management sederhana

### Output
Mampu membangun frontend SPA yang terhubung dengan API.

---

## Minggu 13 – Fullstack Project

### Fokus Materi
- Integrasi frontend & backend
- Login
- API integration
- Deployment

### Output
Membuat aplikasi fullstack sederhana.

---

## Minggu 14 – Deployment & DevOps

### Fokus Materi
- Konsep deployment aplikasi web (frontend & backend)
- Deployment frontend ke Vercel/Netlify
- Deployment backend ke Render/Railway
- Docker dasar (opsional, pengenalan)

### Output
Aplikasi berhasil live dan dapat diakses publik.

---

## Minggu 15 – Final Project & Presentasi

### Fokus Materi
- Penyempurnaan fitur inti
- Perbaikan UX/UI ringan
- Dokumentasi teknis (README, endpoint, setup)
- Demo dan presentasi proyek

### Output
Proyek fullstack final siap dipresentasikan dengan dokumentasi rapi.
